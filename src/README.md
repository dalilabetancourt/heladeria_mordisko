# 🍦 Heladería Mordisko API

API backend para la gestión de productos y recursos de una heladería.
Este proyecto está desarrollado como práctica de backend utilizando **Node.js** y **Express**.

---

## 📌 Descripción

Este proyecto consiste en una API REST que permitirá gestionar diferentes elementos de una heladería, como por ejemplo:

* Productos
* Sabores de helado
* Categorías
* Rutas de la API

El objetivo principal es aprender a construir una estructura backend organizada y escalable.

---

## ⚙️ Tecnologías utilizadas

* Node.js
* Express
* Nodemon
* Dotenv

## 🚀 Instalación

Clonar el repositorio:

```
git clone https://github.com/dalilabetancourt/heladeria_mordisko.git
```

Entrar en la carpeta del proyecto:

```
cd heladeria_mordisko
```

Instalar dependencias:

```
npm install
```

---

## ▶️ Ejecutar el proyecto

Para iniciar el servidor en modo desarrollo:

```
npm run dev
```

El servidor se ejecutará en:

```
http://localhost:3000
```
